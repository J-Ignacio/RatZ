# RatZ 🧟‍♂️🐀

*Día 1... o tal vez el 100, ya perdí la cuenta.*
Bienvenido a **RatZ**, un juego de supervivencia zombi en 2D donde cada recurso cuenta y la muerte es definitiva.
Aquí, tu única misión es recolectar, fabricar y sobrevivir a las hordas que acechan en la oscuridad. El apocalipsis no perdona errores; si mueres, se acabó todo (permadeath ☠️). ¡Prepárate para saquear y luchar por tu vida! 🎒🔨🩸

## Estructura del Proyecto

Este repositorio contiene el código fuente de RatZ. El desarrollo se divide en dos partes principales:

* **`/python_prototype`**: Prototipo inicial desarrollado en Python utilizando Pygame. Utilizado originalmente para probar mecánicas y lógicas base.
* **`/project_godot/rat-z`**: El proyecto principal y actual, desarrollado en **Godot Engine 4.x**. Aquí es donde el juego toma forma final con su motor definitivo.

## Cómo empezar

Dependiendo de la versión que desees explorar:

**Godot Engine (Principal)**
1. Descarga e instala [Godot Engine 4.x](https://godotengine.org/download).
2. Importa la carpeta `/project_godot/rat-z` en el gestor de proyectos de Godot.
3. Ejecuta el proyecto.

**Prototipo Python**
1. Asegúrate de tener Python 3.x instalado.
2. Instala las dependencias necesarias (como Pygame) que se encuentren en la carpeta del prototipo.
3. Ejecuta el script principal de Python.
