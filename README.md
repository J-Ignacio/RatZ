# 🐀 Ratz

## 🛠️ Características del Proyecto

* **Supervivencia y Scavenging:** Busca recursos para mantenerte con vida.
* **Crafteo:** Crea tus propias armas y herramientas.
* **Muerte Permanente (Permadeath):** Si caes, pierdes todo tu progreso.
* **Acción 2D:** Plataformas y mecánicas de movimiento dinámicas.

## 📂 Estructura del Repositorio

Actualmente el proyecto cuenta con dos implementaciones en diferentes etapas de desarrollo:

### 1. Prototipo en Python (`/python_prototype`)
Un prototipo inicial para probar las mecánicas básicas de movimiento y físicas (gravedad, saltos).
* **Tecnología:** Python con la librería `pygame`.
* **Cómo ejecutarlo:**
  1. Asegúrate de tener Python instalado.
  2. Instala las dependencias: `pip install pygame`
  3. Ejecuta el archivo principal: `python python_prototype/main.py`

### 2. Proyecto en Godot (`/project_godot/rat-z`)
El proyecto principal y oficial del juego en desarrollo activo usando un motor de videojuegos moderno.
* **Tecnología:** Godot Engine (GDScript).
* **Cómo ejecutarlo:**
  1. Descarga e instala Godot Engine.
  2. Abre Godot e importa el archivo `project.godot` ubicado en la carpeta `project_godot/rat-z/`.
  3. Presiona Play para iniciar el juego.
